# Python贪吃蛇游戏

一个使用 Python 和 Pygame 制作的经典贪吃蛇游戏。

## 特性

- 多种难度级别
- 流畅的动画效果
- 背景音乐和音效
- 支持键盘控制
- 在线游玩支持

## 如何游玩

访问 [游戏页面](https://knightmaro.github.io/) 即可开始游戏。

### 控制方式
- ↑ 向上移动
- ↓ 向下移动
- ← 向左移动
- → 向右移动
- P 暂停游戏

## 本地运行

1. 克隆仓库
2. 安装依赖：`pip install pygame`
3. 运行：`python main.py`
